2. Fundamental Concepts

2.1 What is an Algorithm?

An algorithm is a finite sequence of well-defined instructions to solve a specific problem or perform a computation.

Definition 2.1 (Algorithm)

An algorithm is a step-by-step procedure that:

  1. Takes an input
  2. Performs a series of operations
  3. Produces an output
  4. Terminates after a finite number of steps

2.2 Algorithm Example

Here's a simple algorithm to find the maximum element in an array: 

function findMax(array):
    if array is empty:
        return null
    
    max = array[0]
    for each element in array:
        if element > max:
            max = element
    
    return max

3. Practical Examples

3.1 Binary Search Implementation

Binary search is an efficient algorithm for finding an item in a sorted array. It works by repeatedly dividing the search interval in half. 

function binarySearch(array, target):
    left = 0
    right = array.length - 1
    
    while left <= right:
        mid = floor((left + right) / 2)
        
        if array[mid] == target:
            return mid
        else if array[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1  // not found

3.2 Analysis

  • Time Complexity: O(log n)
  • Space Complexity: O(1)
  • Prerequisite: Array must be sorted
üí° Key Insight

Binary search achieves logarithmic time by eliminating half of the remaining elements at each step. This makes it vastly superior to linear search for large datasets.
